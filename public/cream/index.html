GETTING STARTED / QUICK START

Quick start

Get Thunderbolt running on your machine and talking to your first agent in under five minutes. No account, no server — everything runs on your device.

Install

# {{ osComment }}
{{ installCmd }}

Or download the app directly from thunderbolt.io/download — builds are available for macOS, Windows, Linux, iOS, and Android.

Connect an agent

Thunderbolt speaks to any ACP-compatible agent. Add one from the command line, or from Settings → Agents in the app:

$ tb agent add claude-code
✓ claude-code connected (acp v1.2)
$ tb chat --agent claude-code
No API key handling. Thunderbolt never proxies your credentials — keys are stored in your OS keychain and sent only to the endpoint you configure.

Optional: sync across devices

Out of the box Thunderbolt is fully local. To sync history and settings across devices, point every client at a sync server you control:

01
Run the server

One container: docker run thunderbolt/sync. Works on-prem, in your cloud, or air-gapped.

02
Point your clients at it

Set the server URL in Settings → Sync on each device. End-to-end encrypted by default.

03
That's it

Conversations, agents, and skills follow you across desktop, web, and mobile.
