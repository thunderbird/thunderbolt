# Thunderbolt AI Agents Directory

This directory contains tool-agnostic resources for AI development assistance on the Thunderbolt project. These files are designed to work with any AI coding assistant (Claude Code, Cursor, GitHub Copilot, etc.) while maintaining consistency and project context.

## Directory Structure

```
.agents/
├── context/              # Condensed context files for quick AI reference
├── analysis/            # Detailed project analysis results  
├── tasks/               # Reusable analysis task definitions
├── commands/            # Custom development commands
├── agents/              # Specialized agent configurations
├── config/              # Tool-specific configurations
└── README.md           # This file
```

## Context Files (`./context/`)

**Purpose**: Condensed, AI-optimized summaries for efficient context loading

- **`architecture.md`** - Technology stack, project structure, and core patterns
- **`patterns.md`** - API interfaces, code conventions, and implementation patterns

## Usage Guidelines

### For AI Assistants
1. **Start with Context**: Reference `./context/` files for quick project understanding
2. **Deep Analysis**: Use `./analysis/` for comprehensive information on specific areas
3. **Task Execution**: Run tasks from `./tasks/` to generate updated analysis
4. **Commands**: Use `./commands/` for project-specific development workflows

### For Developers
1. **Context Updates**: Run analysis tasks manually to refresh context when needed
2. **Custom Commands**: Add project-specific commands to `./commands/`
3. **Agent Development**: Define specialized agents in `./agents/`

### Context Refresh Process
1. Run Task 1 (architecture) first - **required**
2. Run Tasks 2-8 in parallel for efficiency
3. Run Task 9 (synthesis) to compile results
4. Run Task 10 (validation) to verify effectiveness
5. Update condensed context files from analysis results

## Integration with Development Tools

### Claude Code
- Automatically recognizes `.agents/` structure when referenced in `CLAUDE.md`
- Uses context files for enhanced project understanding
- Executes custom commands through integrated terminal

### Other AI Tools
- Generic structure works with any tool that can read markdown and execute scripts
- No vendor lock-in - easily adaptable to new AI coding assistants
- Self-documenting through comprehensive README files

## Maintenance

This directory is maintained through:

- **Team Collaboration**: Shared resource for consistent AI assistance across team members
